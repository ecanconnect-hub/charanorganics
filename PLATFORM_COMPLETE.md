## 🚀 **HOW TO USE ADMIN PANEL**

### **Step 1: Become Admin**

1. Sign up on website: http://localhost:3000/signup
2. Go to Supabase SQL Editor
3. Run:
```sql
UPDATE profiles 
SET role = 'admin' 
WHERE email = 'your-email@example.com';
```

### **Step 2: Access Admin Panel**

Visit: http://localhost:3000/admin

### **Step 3: Add Products**

1. Click "Manage Products"
2. Click "+ Add New Product"
3. Fill in the form:
   - Product ID (e.g., PROD001)
   - Title (English & Telugu)
   - Description
   - MRP & Selling Price
   - Stock Quantity
   - Shipping Charges
   - Image URL (optional)
   - Specifications
   - Usage instructions
4. Click "Add Product"

### **Step 4: Edit Products**

1. Go to Products page
2. Click "Edit" on any product
3. Update fields
4. Click "Update Product"

### **Step 5: Manage Orders**

1. Go to Orders page
2. Filter by status
3. Update order status via dropdown

### **Step 6: Verify Payments**

1. Go to Payments page
2. Click "Review" on pending payment
3. View screenshot
4. Click "Verify" or "Reject"
