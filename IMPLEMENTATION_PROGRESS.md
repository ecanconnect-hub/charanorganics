## 🚀 **HOW TO USE**

### For Customers:
1. Visit http://localhost:3000
2. Browse products
3. Add to cart
4. Login/Signup
5. Checkout
6. Pay via UPI
7. Upload payment proof

### For Admin:
1. Set up admin user in database:
   ```sql
   UPDATE profiles 
   SET role = 'admin' 
   WHERE email = 'your-email@example.com';
   ```

2. Visit http://localhost:3000/admin
3. Manage products, orders, payments
